# VERANO ESTATE by Chef Franko

A luxury villa guest management and culinary services website for VERANO ESTATE in Puerto Vallarta, Mexico.

## Overview

This repository contains the web interface for VERANO ESTATE's guest services, including:

- **Guest Information Form**: Comprehensive form for guest details, room assignments, transportation, and special requests
- **Interactive Menu**: Daily breakfast, lunch, and dinner menu selections
- **Event Dashboard**: Wedding and special event planning tools

## Files

- `index.html` - Main landing page with navigation to all services
- `guest-form.html` - Guest information form (Formulario de Información de Huéspedes)
- `HTML-MENU.html` - Interactive daily menu selection
- `untitled-1.html` - Wedding event dashboard (Panel de Control Culinario)

## Features

### Guest Information Form
- Group and contact information
- Airport transportation options (VIP SUV, taxi, etc.)
- Room assignments for up to 11 rooms
- Individual guest check-in/check-out details
- Additional services (Sunday meal service, cleaning service)
- Special activities booking (mariachi, ATV tours, yacht tours, spa, etc.)
- Provisions and beverages ordering
- Dietary restrictions and allergies

### Interactive Menu
- Daily breakfast buffet with eggs of the day
- Lunch selection with chef's suggestions
- Gourmet dinner options
- Allergy and dietary preferences tracking

### Event Dashboard
- Event overview with key metrics
- Day-by-day itinerary
- Meal distribution visualization
- Personalized allergy care plan
